# configure project 
* create project on GitHub
* configure git
`.gitignore`
* clone project on your computer
`git clone ...`
* inititate project
`npm init -y`
* install Cypress
`npm install --save-dev cypress`
* run cypress
`npx cypress open`
* run todo project
`npx cypress run --spec cypress/integration/1-getting-started/todo.spec.js`
